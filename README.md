# 🎂 Anaya's Birthday Celebration

A beautiful, interactive birthday greeting page created with HTML, CSS, and JavaScript.

## Features
- ✨ Animated falling petals
- 🎈 Floating balloons
- 💫 Sparkle burst effects
- 🎂 Festive birthday cake
- 💌 Personalized message card
- 🌟 Special wishes grid
- 📱 Fully responsive design

## How to View
1. Open `anaya_birthday.html` in your browser, OR
2. Visit: [GitHub Pages Link] (once enabled)

## Customization
Feel free to edit:
- The name "Anaya" to personalize it
- Colors in the `:root` section
- Messages and wishes
- Emojis and decorations
